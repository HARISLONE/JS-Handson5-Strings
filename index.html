Q.1): You are given a string S, and your task is to return an array B ( 2having a size of , where )[B]0B contains the count of character A (uppercase) in string S and 1[contains the count of character D (uppercase) in string S. ]

Ans: 
             
                const countCharacters = (S) => {

                    var B = [2];
                    let countA = 0;
                    let countD = 0;
                    let i;
                    for (i = 0; i < S.length; i++) {
                      if (S[i] === "A") {
                        countA += 1;
                      } else if (S[i] === "D") {
                        countD += 1;
                      }
                    }
                    B[0] = countA;
                    B[1] = countD;
                    return B;
          };

Q.6): You are given a string S consisting of two letters A and D,where each character represent the winner of N games played between Aditya and Danish, where letter A represents the win of Aditya and letter D represents the win of Danish. You need to find out the that which player wins the maximum number of games or there is a draw between them.

Ans: 
             
                var Game_Winner = (S) => {
            
                    let count = 0;
                    let add = 0;
                    for(let i = 0; i < S.length; i++)
                    {
                      if(S[i] == "A")
                      {
                        count++;
                      }
                      else{
                        add++;
                      }
                    }
                    if(count == add)
                    {
                      return "Draw";
                    }
                    else if(count > add){
                      return "Aditya";
                    }
                   else{
                      return "Danish";
                    }
                  };

Ans: 
             
                var Palin_Check = (str) =>
                {
                	const len = str.length;

                            for (let i = 0; i < len / 2; i++) {
                            
                                if (str[i] !== str[len - 1 - i]) {
                                    return 'False';
                                }
                            }
                            return 'True'; 
                };
